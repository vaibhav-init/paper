# Paper

Responsive Docs app, compatible with Android and Web platforms.

### Features
- Google Authentication
- State Persistence
- Link sharing
- Auto saving
- Collaborative Editing using SocketIo


### Project Structure


```
lib/

|- common/
|- constants/
|- models/
|- repository/
|- views/
|- main.dart
```





### Tech Stack
**Backend**: Nodejs, Express, MongoDB, Mongoose, SocketIO

**Frontend**: Flutter

**State Management**: Riverpod
